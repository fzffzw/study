##mysql主从复制原理图
  Master				Slave
|---------------|		|---------------------------------
| Data changes	|		|				 |
|	|	|		| I/O				 |
|	|	|		| thread			 |
|     binary------------------------------>			 |
|	log	|		|    	|			 |
|		|		| write |			 |
|		|		|	|			 |
|----------------		|    relay------->read  -| sql   |
				|     log<--------replay-| thread|
				|--------------------------------

一般增删改走主服务器，查询走从服务器
